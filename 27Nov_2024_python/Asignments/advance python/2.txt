program pauses and waits for the user to type something.

2.Converting user input into different data types (e.g., int, float, etc.).
We can convert user input into different data types using the following functions:
- int() function to convert a string into an integer.
- float() function to convert a string into a floating point number.
- str() function to convert a variable into a string.
- bool() function to convert a variable into a boolean value.
- complex() function to convert a string into a complex number.
